# 문제 024. 신기한 소수 찾기

![문제024](./img/문제024_신기한소수.png)

### 문제 분석

- 재귀함수에 자릿수 개념을 붙여 구현
- 자릿수가 한 개인 소수는 2, 3, 5, 7이므로 이 수부터 탐색을 시작한다. 이어서 자릿수가 두 개는 `현재 수 * 10 + a` 를 계산하여 판단하고, 소수라면 재귀 함수로 자릿수를 하나 늘린다. 단, a가 짝수인 경우는 항상 2를 약수로 가지므로 가지치기로 a가 짝수인 경우를 제외한다. 이런 방식으로 자릿수를 N까지 확장했을 때 그 값이 소수라면 해당 값을 출력

### 문제집 풀이

```java
메모리 17608KB, 시간 740ms

public class P2023_신기한소수 {

	static int N;

	public static void main(String[] args) {
		Scanner sc = new Scanner(System.in);
		N = sc.nextInt();

		// 일의 자리 소수는 2, 3, 5, 7이므로 4개 수에서만 시작
		DFS(2, 1);
		DFS(3, 1);
		DFS(5, 1);
		DFS(7, 1);
	}

	private static void DFS(int number, int jarisu) {
		if(jarisu == N) {
			if(isPrime(number)) {
				System.out.println(number);
			}

			return;
		}

		for(int i=1; i<10; i++) {
			if(i % 2 == 0) {		// 짝수라면 더 이상 탐색할 필요가 없음
				continue;
			}

			if(isPrime(number * 10 + i)) {	// 소수라면 재귀 함수로 자릿수를 늘림
				DFS(number*10+i, jarisu+1);
			}
		}
	}

	private static boolean isPrime(int num) {
		for(int i=2; i<=num/2; i++) {
			if(num % i == 0) {
				return false;
			}
		}

		return true;
	}
}
```
