# 06-4. 이진 탐색

### 이진 탐색 (binary search)

- 데이터가 정렬돼 있는 상태에서 원하는 값을 찾아내는 알고리즘
- 대상 데이터의 중앙값과 찾고자 하는 값을 비교해 데이터의 크기를 절반씩 줄이면서 대상을 찾는다.
- 시간 복잡도 : $O(logN)$

<br />

### 이진 탐색 과정

오름차순으로 정렬된 데이터에서 아래 4가지 과정을 반복 (내림차순이라면 조건을 반대로 하여 과정을 반복)

1. 현재 데이터셋의 중앙값을 선택
2. '중앙값 > 타깃 데이터' 일 때 중앙값 기준으로 왼쪽 데이터셋을 선택
3. '중앙값 < 타깃 데이터' 일 때 중앙값 기준으로 오른쪽 데이터셋을 선택
4. 과정1 ~ 3을 반복하다가 '중앙값 == 타깃 데이터'일 때 탐색을 종료
