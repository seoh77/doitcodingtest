# 08-3. 유클리드 호제법

### 유클리드 호제법(euclidean-algorithm)

- 두 수의 최대 공약수를 구하는 알고리즘
- 일반적으로 최대 공약수를 구하는 방법은 소인수분해를 이용한 공통된 소수들의 곱으로 표현할 수 있지만 유클리드 호제법은 좀 더 간단한 방법을 제시한다.

<br />

### 핵심 이론

- MOD 연산이 최대 공약수를 구하는 데 사용하는 핵심 연산이다.
- MOD 연산 : 두 값을 나눈 나머지를 구하는 연산

<br />

### MOD 연산으로 구현하는 유클리드 호제법

1. 큰 수를 작은 수로 나누는 MOD 연산을 수행한다.
2. 앞 단계에서의 작은 수와 MOD 연산 결괏값(나머지)으로 MOD 연산을 수행한다.
3. 나머지가 0이 되는 순간의 작은 수를 최대 공약수로 선택한다.
