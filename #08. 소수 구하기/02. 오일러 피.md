# 08-2. 오일러 피

### 오일러 피 함수 $P[N]$의 정의

1부터 N까지 범위에서 N과 서로소인 자연수의 개수

<br />

### 오일러 피 함수의 원리

1. 구하고자 하는 오일러 피의 범위만큼 배열을 초기화한다.
2. 2부터 시작해 현재 배열의 값과 인덱스가 같으면(= 소수일 때) 현재 선택된 숫자(K)의 배수에 해당하는 수를 배열 끝까지 탐색하며 $P[i] = P[i] - P[i]/k$ 연산을 수행한다. (i는 K의 배수)
3. 배열의 끝까지 2번을 반복하여 오일러 피 함수를 완성한다.
